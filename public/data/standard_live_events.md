# 重要事件池及解读｜标准版

## 一、已经发生的事件

### 1. 特斯拉Q2财报暴雷：以价换量+AI烧钱，自由现金流两年来首转负

**发生了什么**
特斯拉Q2营收282.4亿美元(+26%)超预期，交付48万辆(+25%)历史最高Q2。但毛利率16.8%远低于预期19.4%，营业利润3.98亿美元(-57%)，营业利润率仅1.4%，调整后EPS 0.33 vs 预期0.52。自由现金流-10.9亿美元（两年来首次转负），资本开支57.89亿美元(+142%)。FSD活跃订阅148万(+56%)，Cybercab已开始生产。全年Capex预计超250亿美元，接近2025年的三倍。股价单日-14.52%。

**为什么值得重视**
这是市场对"AI投资故事"定价逻辑转变的标志性事件。此前市场愿意给AI概念公司"烧钱换未来"的估值溢价，但特斯拉的财报显示——以价换量侵蚀毛利（毛利率从21%降至16.8%）+AI/Cybercab/Optimus全面烧钱=FCF转负。市场不再接受"增长故事"替代"盈利验证"。这对所有高估值、高Capex的科技公司形成估值参照压力。

**行业影响**
- 电动车产业链：以价换量模式下的利润率压力将传导至整个产业链，电池/材料环节可能比整车更具议价能力
- AI算力链：特斯拉Capex+142%是AI需求的验证，但市场在区分"烧钱方"和"赚钱方"
- FSD/Robotaxi：FSD订阅增长(+56%)和Cybercab投产是积极信号，但商业化节奏仍是关键

**周期与主线判断**
偏短期冲击——特斯拉单季财报不代表产业趋势拐点，48万辆交付创纪录说明需求没坏。但市场情绪需要时间消化"利润率下滑+Capex暴增"的组合。中期看，如果FSD和Robotaxi能在2026H2产生规模收入，叙事会修复。

**验证项**
- 7-8月特斯拉中国/欧洲月度交付数据
- Q3毛利率能否企稳
- FSD在更多市场的监管审批进度

### 2. 谷歌AI豪赌：云+82%但自由现金流数十年来首转负

**发生了什么**
谷歌Q2营收1198亿(+24%)，云营收248亿(+82%)大超预期，营业利润408亿(+30%)。云积压订单5140亿美元首破5000亿。但资本开支449亿美元(同比翻倍)，自由现金流-59亿美元首次转负，全年Capex指引从1800-1900亿上调至1950-2050亿，预计2027年仍将大幅增长。6月完成496亿美元股权融资+203亿美元债券。股价-7.13%。

**为什么值得重视**
这是本轮AI投资周期的"压力测试"结果。谷歌云+82%证明AI需求真实且巨大，但Capex翻倍+FCF转负说明供给端（算力建设）的投入速度远超收入确认速度。市场给出的答案：即使需求超预期，只要烧钱无节制，估值就要打折。这个逻辑将直接影响后续微软/亚马逊/Meta财报的市场反应。

**行业影响**
- AI基础设施链：Capex上修验证了算力需求持续增长，利好芯片/设备/光通信等环节
- 云计算行业：谷歌云与AWS差距缩小（份额21% vs 28%），AI正重新定义云竞争格局
- AI应用层：云积压订单5140亿说明企业端AI部署在加速，利好AI应用和SaaS公司

**周期与主线判断**
中期结构性利好AI基础设施链——谷歌的Capex上修和2027年继续增长的表态，为AI硬件需求提供了2年以上的确定性。但短期市场情绪受FCF转负压制。核心矛盾："需求在加速，但投资回报的验证窗口还没到"。

**验证项**
- 微软/亚马逊Q2财报中的AI收入拆分和Capex指引
- Q3谷歌云收入增速能否维持80%+
- 谷歌Capex中GPU/TPU采购的具体节奏

### 3. 中东局势急剧升级：布油突破100美元，美联储加息预期跳升

**发生了什么**
胡塞武装宣布对沙特实施"海上禁运"并袭击两艘沙特油轮。特朗普称"认真考虑"重启对伊朗的大规模打击（"规模空前"）。美国向中东增派F-16/F-35中队、两个航母打击群（共17艘舰船）、医护人员和武器装备。布油+7%至100.69美元/桶（5月22日以来首次破100），WTI+6%至92.19美元/桶。库欣原油库存接近"罐底"。美联储9月加息概率从52%跳升至80%+。10年期美债收益率4.714%（2025年1月以来最高），2年期4.370%。

**为什么值得重视**
这是继美伊6月备忘录后的第二次重大升级，且烈度更高——从"两国军事对峙"质变为"全球航运通道封锁风险"。霍尔木兹海峡承担全球约34%原油海运量，如果持续封锁，油价可能冲击120-150美元（高盛/中信建投预测）。油价→通胀→加息→美债收益率→全球风险资产估值的传导链正在自我强化。当前库欣库存极低意味着市场几乎没有缓冲空间吸收供应中断。

**行业影响**
- 能源板块：直接利好石油/天然气/油服/油运公司
- 新能源：高油价加速能源转型需求，利好风光储
- 有色金属/化工：油价推高生产成本，但需求端受益于通胀预期
- 科技板块：利率上行压制高估值成长股，特别是尚未盈利的AI公司
- 军工/国防：美国国防航空股大涨（洛克希德马丁+10.54%/雷神+7.33%）

**周期与主线判断**
短期强烈看多能源/军工，看空高估值科技。中期取决于冲突持续时间——若2-4周内外交解决，油价回落后科技股修复；若持续封锁超过1个月，全球宏观环境将发生结构性变化（滞胀风险上升），需要全面调整组合配置。

**验证项**
- 美伊外交接触信号（是否恢复谈判）
- 红海/霍尔木兹实际通航情况
- 7/28-29美联储会议声明中对通胀的描述变化
- 美国汽油零售价是否突破4.5美元/加仑

### 4. 证监会稳市场+央���MLF投放：国内政策底信号

**发生了什么**
证监会7/23召开党建暨监管工作座谈会，提出"综合施策全力维护市场平稳运行"、"推动中长期资金稳步提升入市规模和比例"、"加强应对全球市场波动和风险跨境传导的政策储备"、"市场整体具备较好的配置价值"。央行7/24开展5000亿MLF操作（到期4000亿，净投放1000亿），连续第三个月加量续作，且提前至24日落地。

**为什么值得重视**
"市场整体具备较好的配置价值"是证监会会议中首次使用的正面定性表述，叠加MLF连续第三个月加量+提前投放，国内政策底信号非常明确。在外部冲击（油价100+美股暴跌）加大的背景下，政策层提前释放稳市场信号，意图对冲海外风险传导。这对A股的底线思维有重要意义——有政策兜底，系统性风险可控。

**行业影响**
- 券商/金融：直接受益于稳市场政策和流动性投放
- 整体A股：降低系统性风险溢价，支撑估值底部

**周期与主线判断**
政策底确认，但市场底需时间。历史经验显示政策底到市场底通常间隔1-3周，且需要外部冲击消化。当前处于政策底区间，不宜过度悲观。

**验证项**
- 后续是否有更具体的入市资金安排（中长期资金入市方案）
- 两融余额和北向资金流向变化
- 上市公司回购/增持公告的持续性

### 5. 日韩今早暴跌：亚太市场开盘恐慌

**发生了什么**
7/24日经225开盘-1.26%后暴跌超1600点(-2.42%)失守65000点。KOSPI开盘-2.17%后跌幅扩大至3%+失守6900点。SK海力士-4%，三星电子-3%，软银集团-5.61%，爱德万测试-6.1%。韩国金融服务委员会将单一个股杠杆ETF存款要求（3000万韩元现金）提前至7/31实施。日元对美元跌至163.96再创新低。

**行业影响**
- 对A股科技板块：情绪传导（低开），但A股科技昨天已经提前调整
- 对存储芯片：SK海力士-4%但隔夜ADR+2.6%，韩股更多是本土资金行为

**周期与主线判断**
短期情绪冲击，不改变韩股去杠杆进程和全球半导体需求逻辑。

**验证项**
- 日韩收盘跌幅是否收窄
- 韩国是否有进一步的救市措施

### 6. VC（碳酸亚乙烯酯）价格暴涨：新能源上游供需缺口

**发生了什么**
7/23 VC均价报20万元/吨，单日+11.11%，一周+21%，时隔4年多重上20万。驱动：动力电池新国标(7月实施)要求热失控后2小时不起火→电池厂增加VC添加量；储能排产7月283GWh环比+5.6%（储能VC添加比例>3%，高于动力电池的1-3%）；行业库存极低（多数厂商仅1周原料库存）；宁德时代提前锁单（三年供应9万吨VC），比亚迪入股VC大厂华盛锂电。

**为什么值得重视**
VC涨价是新能源上游供需矛盾的缩影——政策驱动（新国标）+需求超预期（储能）+供给刚性（危化品审批严/建设周期2-3年）+巨头锁单=价格暴涨。这不仅是VC一个品种的故事，反映的是新能源产业链从"产能过剩"到"局部紧缺"的结构性变化。宁德时代和比亚迪亲自下场锁原料，说明了问题的紧迫性。

**行业影响**
- 电解液添加剂：VC/FEC生产企业直接受益（华盛锂电/富祥股份/永太科技/天赐材料/新宙邦）
- 电解液：VC成本上升可能传导至电解液价格
- 储能：需求增长持续超预期，利好储能产业链

**周期与主线判断**
中期看多——新国标落地+储能增长+供给刚性=VC紧缺至少持续到2026Q4（中信建投测算有效供给11.5万吨 vs 需求持续增长）。但短期股价涨幅较大(相关概念股昨日涨停)，追高风险增加。

**验证项**
- 7-8月VC价格能否维持20万以上
- 新增产能投放进度（华盛锂电6万吨项目/富祥技改）
- 电解液厂商的VC库存天数变化

## 二、即将发生的事件

### 7/27（周一）长鑫科技科创板上市

**将验证什么**
- A股对存储国产替代龙头的定价热情
- 上市首日流动性虹吸效应对科技板块的影响程度
- DRAM产业链的估值参照——长鑫上市后，设备/材料/封测等环节的估值空间将被重新定义

**可能结果及影响**
- 若上市首日大涨：存储/半导体国产替代情绪升温，利好设备/材料/封测链
- 若上市首日表现平淡或破发：短期打击存储国产化信心，但定价合理后反而提供配置窗口
- 无论首日如何：上市前1-2天科技板块可能有流动性虹吸压力（参考中芯上市经验），但本轮科技已提前调整，"提前消化"效应可能降低冲击幅度

### 7/28-29 美联储议息会议

**将验证什么**
- 7月是否加息（市场定价36%概率，经济学家一致预期不动）
- 沃什对通胀的最新定调——是否因油价破100而转鹰
- 9月加息的前瞻暗示——点阵图/声明措辞变化

**可能结果及影响**
- 若维持利率+声明偏鸽：市场短期修复，科技股反弹
- 若维持利率+声明偏鹰（提示9月加息）：美债收益率继续上行，科技股进一步承压

- 各家Capex指引是否跟随谷歌大幅上修
- 市场对"AI投资回报"的定价逻辑是否延续特斯拉/谷歌模式

**可能结果及影响**
